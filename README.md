🤖 AI-Style ChatBot

This is a simple AI-style chatbot web application built using HTML, CSS, and Vanilla JavaScript.
The chatbot simulates a conversational interface where users can type messages and receive automatic responses based on keywords.

It demonstrates DOM manipulation, event handling, and simple chatbot logic using JavaScript.

🔧 Tech Stack

💻 HTML5 – Structure of the chatbot interface
🎨 CSS3 – Styling and responsive layout using Flexbox
⚙️ JavaScript (ES6+) – Chatbot logic and DOM manipulation

💻 Features

💬 Chat interface similar to messaging apps
📜 Scrollable chat history
⌨️ Send messages using Send button or Enter key
🤖 Automatic chatbot replies based on keywords
⏳ Small delay to simulate AI typing response
📱 Responsive design using CSS Flexbox

💬 Built-in Bot Commands

Try typing these keywords in the chat:

👋 hello / hi / hey → Greets the user
🙂 how are you → Bot asks how the user is doing
🙏 thank you / thanks / thank u → Bot replies politely
💰 price / cost → Shows pricing information
📚 course / classes → Displays available courses
🤖 name → Bot introduces itself
❓ help → Shows supported commands

If the chatbot doesn't understand a message, it gives a fallback response explaining its limitations.
